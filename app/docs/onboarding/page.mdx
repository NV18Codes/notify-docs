# Customer Onboarding

Set up Notifyy to start sending WhatsApp messages.

👉 [Go to Notifyy Dashboard](https://www.notifyy.io/)

---

## Step 1: Connect WhatsApp

Go to Settings → WhatsApp → Connect Account

<img src="/docs/onboarding/step-1.png" className="rounded-lg border my-4" />

### Steps

1. Click Connect Account
2. Enter business details
3. Add phone number
4. Complete OTP verification

⚠️ Your number must not be linked to another WhatsApp API.

---

## Step 2: Verify Business

<img src="/docs/onboarding/step-5.png" className="rounded-lg border my-4" />

### Steps

1. Enter business details
2. Submit documents
3. Wait for approval

⏱️ Takes a few minutes to a few hours

---

## Step 3: Create Template

<img src="/docs/onboarding/step-9.png" className="rounded-lg border my-4" />

### Steps

1. Go to Templates
2. Click Create Template
3. Add message
4. Submit

⚠️ Template must be approved before sending campaigns

---

## Step 4: Import Contacts

<img src="/docs/onboarding/step-13.png" className="rounded-lg border my-4" />

### Steps

1. Go to Contacts → Import
2. Upload CSV
3. Map fields
4. Save

💡 Include country code in phone numbers

---

## Step 5: Send First Campaign

<img src="/docs/onboarding/step-18.png" className="rounded-lg border my-4" />

### Steps

1. Go to Campaigns
2. Select template
3. Choose audience
4. Click Send

💡 Start with small audience

---

## Common Issues

- Template not approved → Cannot send
- Wrong phone format → Messages fail
- API not connected → Campaign disabled
