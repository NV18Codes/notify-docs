# कोडलेस चैटबॉट बिल्डर

बिना बैकएंड कोड के बातचीत फ़्लो बनाएँ — ज़रूरत पड़ने पर मानव हैंडऑफ करें।

## ब्लॉक

- **ट्रिगर:** कीवर्ड, टेम्पलेट उत्तर, QR स्कैन।
- **शाखाएँ:** CRM विशेषताएँ या उपयोगकर्ता इनपुट।
- **क्रियाएँ:** टेम्पलेट भेजें, फ़ील्ड अपडेट, टिकट बनाएँ।

## विश्लेषण

प्रति चरण ड्रॉप-ऑफ और मानव टेकओवर दर देखें।
