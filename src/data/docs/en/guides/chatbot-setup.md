# Chatbot setup guide

Ship a production-ready assistant in an afternoon.

![Analytics and funnels help you tune bot steps after go-live](/docs/images/analytics-dashboard.jpg "Dashboard-style analytics — use Notifyy’s built-in reports to refine bot drop-off.")

## 1. Map intents

List the top 10 reasons customers message you — order status, refunds, bookings. For each intent, define **training phrases** and **success criteria**.

## 2. Build the flow

Use the **visual builder** to chain steps:

- Collect order ID → call API → render template with tracking link.
- If confidence is low → **handoff** to live chat with transcript.

## 3. Content & tone

Keep messages short; use quick replies instead of open-ended questions when possible.

## 4. Test

Run **sandbox tests** with real phone numbers. Validate multilingual paths if you serve Hindi or Kannada audiences.

## 5. Publish & monitor

Version your flow, publish, and watch **per-step drop-off** in analytics. Iterate weekly.
