# WhatsApp Business API

Notifyy आधिकारिक **WhatsApp Business Platform** पर आधारित है — Meta द्वारा प्रलेखित क्लाउड API संगत।

## क्षमताएँ

- संदेश, मीडिया और स्थिति के लिए **क्लाउड API** अनुकूल एंडपॉइंट।
- टेम्पलेट जीवनचक्र: ड्राफ़्ट → लंबित → स्वीकृत।
- **वेबहुक** — डिलीवरी रसीदें, उपयोगकर्ता उत्तर, त्रुटियाँ।

## अनुपालन

ऑप्ट-इन, शांत घंटे और आवृत्ति सीमाएँ वर्कस्पेस स्तर पर लागू की जा सकती हैं।
