# बल्क अभियान

स्वीकृत **टेम्पलेट संदेश** खंडित दर्शकों को थ्रॉटलिंग, पुनःप्रयास और विश्लेषण के साथ भेजें।

## दर्शक

टैग, CRM फ़ील्ड या अंतिम इंटरैक्शन से फ़िल्टर करें। अनसब्सक्राइब उपयोगकर्ता स्वतः बाहर।

## रणनीति

**अनुकूल थ्रूपुट** गुणवत्ता रेटिंग सुरक्षित रखता है। A/B परीक्षण उद्यम योजनाओं पर उपलब्ध।

## मेट्रिक्स

**भेजा → डिलीवर → पढ़ा → क्लिक** — अच्छी टारगेटिंग पर **50–70% क्लिक** संभव।
