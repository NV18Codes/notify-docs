# चैटबॉट सेटअप गाइड

उत्पादन-तैयार सहायक जल्दी लॉन्च करें।

## 1. इरादे मैप करें

शीर्ष 10 ग्राहक प्रश्न — प्रशिक्षण वाक्य और सफलता मानदंड परिभाषित करें।

## 2. फ़्लो बनाएँ

दृश्य बिल्डर से चरण जोड़ें: डेटा एकत्र → API → टेम्पलेट; कम विश्वास पर **लाइव चैट हैंडऑफ**।

## 3. परीक्षण

सैंडबॉक्स में वास्तविक नंबरों से परीक्षण करें; हिंदी/कन्नड़ पथ सत्यापित करें।

## 4. प्रकाशित करें

संस्करणित करें, प्रकाशित करें, **ड्रॉप-ऑफ** देखें।
