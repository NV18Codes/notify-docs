# ಕೋಡ್‌ರಹಿತ ಚಾಟ್‌ಬಾಟ್ ಬಿಲ್ಡರ್

ಬ್ಯಾಕೆಂಡ್ ಕೋಡ್ ಇಲ್ಲದೆ ಸಂವಾದ ಪ್ರವಾಹಗಳನ್ನು ರಚಿಸಿ — ಅಗತ್ಯವಿದ್ದಾಗ ಮಾನವರಿಗೆ ಹಸ್ತಾಂತರಿಸಿ.

## ಬ್ಲಾಕ್‌ಗಳು

- **ಟ್ರಿಗರ್‌ಗಳು:** ಕೀವರ್ಡ್, ಟೆಂಪ್ಲೇಟ್ ಉತ್ತರ, QR.
- **ಶಾಖೆಗಳು:** CRM ಗುಣಲಕ್ಷಣಗಳು ಅಥವಾ ಇನ್‌ಪುಟ್.
- **ಕ್ರಿಯೆಗಳು:** ಟೆಂಪ್ಲೇಟ್, ಕ್ಷೇತ್ರ ಅಪ್‌ಡೇಟ್, ಟಿಕೆಟ್.
