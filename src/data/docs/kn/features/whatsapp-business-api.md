# WhatsApp Business API

Notifyy ಅಧಿಕೃತ **WhatsApp Business Platform** ಮೇಲೆ ನಿರ್ಮಿತವಾಗಿದೆ — Meta ಡಾಕ್ಯುಮೆಂಟ್ ಮಾಡಿದ ಕ್ಲೌಡ್ API ಯೊಂದಿಗೆ ಹೊಂದಿಕೆಯಾಗುತ್ತದೆ.

## ಸಾಮರ್ಥ್ಯಗಳು

- ಸಂದೇಶ, ಮಾಧ್ಯಮ ಮತ್ತು ಸ್ಥಿತಿಗೆ API.
- ಟೆಂಪ್ಲೇಟ್ ಜೀವನಚಕ್ರ ಸಿಂಕ್.
- **ವೆಬ್‌ಹುಕ್** — ವಿತರಣಾ ರಸೀದಿಗಳು, ಉತ್ತರಗಳು, ದೋಷಗಳು.
